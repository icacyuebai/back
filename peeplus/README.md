<h1 align="center">数据标注平台</h1>


- 项目说明
> 1、基于flask和vue打造的前后端分离项目<br>
> 2、前后端只依赖于token<br>

前端环境和依赖
----

- node
- yarn
- webpack
- eslint
- @vue/cli ~3


后端端环境和依赖
----

- python3.9
- flask -- web框架
- flask-script
- flask-socketio
- flask-sqlalchemy   -- 用于数据库操作
- numpy
- flask-debugtoolbar
- mysqlclient
- flask-cors
- pyjwt  -- 用于jwt鉴权
- flask-sqlacodegen
- openpyxl -- 用于解析Excel文件
- xlrd -- 用于解析Excel文件
